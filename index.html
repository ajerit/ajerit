Projects

SIGULAB
Sistema de Gestión de la Unidad de Laboratorios

Sistema de Información desarrollado en equipo para la Unidad de Laboratorios de la Universidad Simón Bolívar. Se encarga del manejo del inventario de todas las sustancias que se utilizan en los Laboratorios del campus. Generación de reportes a entidades legales y manejo de permisos para diferentes tipos de usuarios.
Desarrollado en Web2Py, Marzo 2017.

View Project

SIGPAE-Históricos
Sistema de Gestión de Programas Analíticos de Estudio

Sistema para el archivo y gestión de todos los Programas Análiticos de Estudio de la Universidad Simón Bolívar. Permite la transcripción desde formato PDF a una base de datos donde se guarda el contenido. Permite consultar los Programas guardados y generar reportes globales.
Desarollado en Django, Marzo 2017.

View Project

Retina
Interpretador para el lenguaje Retina

Lenguaje de programación inspirado en Logo, junto al concepto turtle graphics, para la generación de imágenes haciendo uso del formato pbm. El proyecto implementa un Lexer, un Parser y un Analizador de Contexto, para realizar la ejecución del programa y generar las imágenes. Utiliza Racc para generar la gramática del lenguaje.
Desarollado en Ruby, Marzo 2017.
